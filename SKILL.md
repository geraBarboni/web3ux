# web3ux

A human-in-the-loop UX pipeline for Web3 products. Each skill feeds the next, with checkpoints where you make the decisions.

## Pipeline

```
/web3-context    →  learn the project, generate persistent context
/web3-brief      →  research: personas, pain points, competitive analysis
/web3-flow       →  user flows + task flows in Mermaid + decision points
/web3-critique   →  heuristic audit of existing flows or UI
/web3-wireframe  →  low-fi wireframes in Figma
```

## Routing guide

**Starting a new project from scratch?**
→ Start with `web3-context`

**Have context, need research?**
→ Use `web3-brief`

**Have research, need flows?**
→ Use `web3-flow`

**Have something already built, need an audit?**
→ Use `web3-critique`

**Have flows, need wireframes?**
→ Use `web3-wireframe`
